# Countdown & Quotes App

A simple JavaScript web app that shows a countdown timer to New Year 2027, a motivational quotes slider, and a welcome popup.

---

## What This App Does

- Shows a live countdown to **January 1, 2027**
- Displays **motivational quotes** that change automatically every 4 seconds
- Shows a **welcome popup** after 5 seconds of opening the page

---

## Project Files

| File | What it does |
|------|--------------|
| `index.html` | Structure of the webpage |
| `style.css` | Styling and design |
| `script.js` | All the logic and functionality |

---

## Features

### Countdown Timer
- Counts down days, hours, minutes, and seconds to New Year 2027
- Auto starts when page loads
- Start and Pause buttons to control the timer
- Shows a message when time is up

### Quotes Slider
- 6 motivational quotes stored in an array
- Changes quote automatically every 4 seconds
- Next and Previous buttons to switch quotes manually

### Modal Popup
- Appears automatically after 5 seconds
- Shows a welcome message
- Close button to dismiss it

---

## How to Run

1. Download or clone the project
2. Open `index.html` in any browser
3. App will start automatically
